A Tabletop RPG system I'm working on that all about teenangers who are paired up to pilot Mechas to fight Eldritch Monsters from stuff like HP Lovecraft. This system is a combination of the various TTRPG games I really like.

My First Model Swap:

A modding project where I first have to 3D model a character and clothes for them then replace a character in the game with them. Probably will be done with a first person shooter since those seem easier to implement it in. Main reason this is so big is because I have to learn how to blender to make a character model/clothes.
